<div class="detail-body">
  <p>Designed and implemented a custom configuration to integrate Zebra label printers into an existing IBM i environment, navigating compatibility challenges between the two systems. The IBM i natively outputs in Intermec Print Language (IPL) while Zebra printers default to their own print language, so bridging that gap required modifying printer firmware directly and leveraging Zebra's proprietary tools to build a configuration that met the organization's specific requirements. Configured all units for network operation and deployed them across more than twenty service locations, where they continue to be maintained and monitored for performance and integration issues. Developed working documentation for the organization's iSeries application vendor, enabling them to extend Zebra hardware support to other clients as a courtesy off the back of this integration.</p>
